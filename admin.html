Google Apps Script Setup — Step by Step
  1. Go to sheets.google.com → Create a new blank spreadsheet → Name it "FamilyShield Leads"
  2. Copy the Sheet ID from the URL — it's the long string between /d/ and /edit
  3. Go to script.google.com → click New project → delete all existing code
  4. Paste the code below, replace YOUR_SHEET_ID and choose a YOUR_SECRET_TOKEN (any random text)
  5. Click Deploy → New deployment → Web App
  6. Set "Execute as: Me" and "Who has access: Anyone" → click Deploy
  7. Copy the Web App URL and paste it in the Backend Connection field above
  8. Also paste your token in the Secret Token field → click Save Connection
